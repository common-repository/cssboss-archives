== Description ==

A Streamlined page to let users navigate your website categories and posts.

== Installation ==

1. Log into your wp-admin
2. Navigate to Plugins > Add New
3. Either Search for 'CSSBoss Archives' or upload the zip file
4. Follow installation steps through WordPress
5. Use the [cssboss_archive] shortcode on the page you wish to display your new archives. We recommend full width pages.
